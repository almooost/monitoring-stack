## [1.0.0] - 2024-06-17

### Added
- Initial release
- Ansible playbooks for monitoring stack deployment
- Docker Compose configurations for customer sites
- Prometheus configuration templates
- SNMP, Blackbox, and Node exporter setups
- Alertmanager integration
- vmagent for centralized alert forwarding
- Grafana provisioning
- Documentation and examples
- Support for Proxmox, VMware ESXi, HP/Cisco switches, Fortigate/SonicWall firewalls
